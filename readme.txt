* Keep the input file in the same path and name it input.xlsx else change line number 11 in index.js with the file name
* Do not keep multiple worksheets in the input file
* Make sure if any existing outfile present in the same directory then delete it before running the script

* Once done with above points run "npm install" only once
* To get the output file "node index.js"
* The output file will get saved in the same directory with the name as "output.xlsx" and "output.json". Delete both the files before reusing it.
